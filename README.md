## Usage

- Generate new application
```bash
hike new ExampleApplication
```

- Help
```bash
hike help

hike <command>

Commands:
  hike new <AppName>  Generate new application                      [aliases: n]

Options:
  --version  Show version number                                       [boolean]
  --help     Show help                                                 [boolean]

```
